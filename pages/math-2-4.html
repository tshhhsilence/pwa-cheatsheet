Свойства нуля
a + 0 = a; a − 0 = a
Переместительное свойство сложения

Сочетательное свойство сложения
(a + b) + с = a + (b + c) = a + b + c
Свойство вычитания суммы из числа
a − (b + c) = a − b − c
Свойство вычитания числа из суммы
(a + b) − c = (a − c) + b = a + (b − c)

Умножение

От перестановки множителей произведение не меняется.
Переместительное свойство умножения
a × b = b × a
Сочетательное свойство умножения
(a × b) × c = a × (b × c)
Распределительное свойство умножения
(a + b) × c = a × c + b × c
(a − b) × c = a × c − b × c
Свойство нуля: если любое число умножить на 0, получится 0
а × 0 = 0
Свойство единицы: если любое число умножить на 1, получится то же число
а × 1 = a

Деление

При делении 0 на число всегда получается 0
0 ÷ a = 0
При делении любого числа на 1 получается то же число
a ÷ 1 = a
На ноль делить нельзя
а ÷ 0

Cначала действия в ( ), потом остальные:
(a + b) × c
c × (a − b)
(a + b) ÷ c

в конце + и -:
